Distributed Task Scheduler using DAG-based Execution

Objective:
Designed and implemented a distributed task scheduling system that executes tasks based on Directed Acyclic Graph (DAG) dependencies. The system ensures correct task ordering, parallel execution, and real-time monitoring of task progress.

Tools / Techniques Used:
Python, FastAPI, PostgreSQL, Redis, Docker, SQLAlchemy, DAG Scheduling, REST APIs, Distributed Systems, Queue-based Task Execution

Outcome:
Developed a scalable distributed scheduler capable of executing dependent tasks reliably across worker nodes while maintaining task state and execution logs in a centralized database. Implemented DAG progress tracking APIs and automated verification scripts to validate dependency enforcement and successful task orchestration.
